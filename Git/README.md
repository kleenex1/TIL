  1. 분산 버전 관리시스템

# Repository & Commit 
  1. Repository</br>
    - 버전별 프로젝트 모습을 나타낸다.</br>
    - 커밋이 저장되는 곳
  2. Commit</br>
    - 하나의 버젼으로 남기는 동작

# Working directory / Staging area / Repository
  - `Working directory` : 현재 작업(수정)한 파일
  - `Staging area` : git add를 한 파일들이 존재하는 영역(index)
  - `repository` : commit을 하면 저장되는 곳(버전)
